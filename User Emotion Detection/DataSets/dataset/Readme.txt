The dataset is provided as is, for research purposes only. Not for commercial use.

The Tweets.txt file contains two tab-separated columns. The format is:
<TweetID>	<Emotion>

The tweets IDs are provided by Twitter itself, and are of type long (overflow may occur otherwise). 
They can be used to retrieve the content of the tweets, according the service's policy.

The dataset includes in total 2.246 tweets.
